Macro Structure of an HTML Document

    <!DOCTYPE html:5>
    <html>
      <head>
        tags to describe the page itself
        tags to link CSS file
        tags to include JavaScript files
        </head>
      <body>
        Main content goes here.
      </body>
    </html>

HTML Links

HTML links are defined with the a tag:

<a> href="https://www.w3schools.com">This is a link </a>

This is a link

Image Tag

HTML images are defined with the img tag:

<img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">

Button Tag

HTML buttons are defined with the button tag:

<button>Click me</button>

Div (division) Tag

The DIV (division) element is a block element, and will always start on a new line and take up the full width available (stretches out to the left and right as far as it can). Most common and generic container element.

<div>style="border: 1px solid black">Hello World</div>

Hello World
